# Changelog

All notable changes to this project will be documented in this file. See [standard-version](https://github.com/conventional-changelog/standard-version) for commit guidelines.

## [0.1.0] - 2026-01-28

### Features

- Initial release of Mermaid Viewer extension
- Detect and render Mermaid diagrams in code blocks
- Support for flowcharts, sequence diagrams, and other Mermaid diagram types
- Toggle switch to enable/disable rendering per page
- Popup UI for extension settings
- Works on all websites
